== Frequently Asked Questions ==

= How to set Theme Front Page? =
Go to "Settings >> Reading >> Front page displays" select "A static page (select below)" and there set following
"Front Page" - deafult front page for your site
and "Blog Page" - page on which the blog posts will be displayed

= Where is Navigation Menu? How to set a dropdown menu? =
To set up the navigation menu go to "Appearance >> Menus" create or use existing menu as Main Navigation.
To create dropdown menu drag the menu items somewhat right and they will automatically get set as dropdown.

= How to add Featured Boxes on Front Page? =
Go to "Appearance >> Widgets" and add "Featured Box - swiftbiz-lite" widget in "Home Featured Sidebar".

= How to create a full width page? =
1. Go to Pages.
2. Click Add New or Edit an existing page.
3. Select "Page : No Sidebar Template" from "Page Attributes >> Template".

= What is preferred size for Logo Image? =
The preferred size for Logo Image is 270x80 pixels.
